# ExKeyboardDemo
键盘表情切换方案, 支持透明状态栏,支持对话框,支持动画展开.

# 特性
- 兼容`低版本`安卓系统
- 支持最新版`微信`那样的动画
- 支持多`child`布局
- 支持`padding`处理
- 支持`SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN`属性
- 支持`普通输入框`切换到`安全密码输入框`
- 支持布局`preview`
- 内容布局支持`wrap_conent`, `margin`属性.(高度很小时, 默认从Gravity.BOTTOM开始布局)
- 支持`表情布局`恢复.(如:显示键盘之前是表情布局, 那么隐藏键盘后就会显示表情布局)
